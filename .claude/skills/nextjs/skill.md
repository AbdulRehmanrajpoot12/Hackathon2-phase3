# Next.js Development Skill

## Purpose
This skill provides expertise in developing applications using Next.js, a popular React framework for production-ready web applications.

## Capabilities
- Create and configure Next.js applications
- Implement pages and routing using the file-based router
- Develop API routes for server-side functionality
- Handle static and dynamic imports
- Implement server-side rendering (SSR) and static site generation (SSG)
- Configure Next.js settings in next.config.js
- Optimize performance with code splitting and lazy loading
- Handle images and assets efficiently
- Implement internationalization (i18n) features
- Work with Next.js middleware and custom server configurations

## Best Practices
- Follow the file-based routing convention
- Use getStaticProps and getServerSideProps appropriately
- Leverage Next.js Image component for optimization
- Implement proper error handling with custom Error pages
- Use CSS Modules or styled-jsx for component styling
- Optimize bundle size and loading performance
- Implement proper SEO practices with Head component

## Common Commands
- `npx create-next-app` - Initialize a new Next.js project
- `npm run dev` - Start development server
- `npm run build` - Build the application for production
- `npm start` - Start the production server

## File Structure Convention
```
my-nextjs-app/
├── pages/
│   ├── index.js
│   └── api/
├── public/
├── components/
├── styles/
└── next.config.js
```
